# Rice Grain Classification Using Machine Vision and Transfer Learning

## Overview
This project classifies rice grain varieties using Computer Vision, Transfer Learning, and Deep Learning.

## Features
- Rice grain image classification
- MobileNetV2 transfer learning
- YOLO-based grain detection
- Web interface for prediction
- Multiple rice varieties supported

## Technologies
- Python
- TensorFlow / Keras
- OpenCV
- YOLOv8
- Flask

## Dataset
Rice Image Dataset

## How to Run
pip install -r requirements.txt
python app.py
