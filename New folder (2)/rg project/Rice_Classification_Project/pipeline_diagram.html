Key Techniques & Design Decisions

Perceptual Hashing — Removes near-duplicate images using average-hash + Hamming distance, preventing the model from memorising repeated samples.
Laplacian Blur Detection — Images with variance < 80 are discarded, ensuring only sharp, informative grains enter training.
Two-Phase Transfer Learning — Phase 1 trains only the head (fast convergence); Phase 2 fine-tunes top base layers (higher accuracy).
Label Smoothing (0.05) — Prevents overconfident predictions and improves generalisation to unseen real-world images.
tf.data + AUTOTUNE — Parallel image loading and prefetching eliminates I/O bottlenecks, maximising GPU utilisation.
3-Tier Confidence System — High / Medium / Low tiers replace binary rejection, giving users helpful feedback instead of hard errors.
Aspect-Ratio Shape Override — AR > 3.5 → Basmati, AR < 1.6 → Arborio. Morphological signal corrects ML errors on clear cases.
Stratified 70/15/15 Split — Each split preserves class proportions, ensuring fair evaluation across all 5 rice varieties.
